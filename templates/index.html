Por que usar?

Devemos usar redes complexas quando tem-se a possibilidade de abstrair ideias vistas no mundo real, para um modelo puramente computacional.

Comunidades

São vértices em uma Rede Complexa que possuem uma alta tendência de se consolidar como uma rede além da que está instaurada.

Reciprocidade

É a tendência de vértices possuírem uma ligação mútua, quando dois vértices apontam para o outro, quando isso ocorre, há uma relação recíproca.
