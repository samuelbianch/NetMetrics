ConectaIF

Este é um projeto de Trabalho de Conclusão de Curso para a obtenção do diploma de Engenheiro da Computação do estudante Samuel Bianch. O projeto visa facilitar as pesquisas com redes complexas para pessoas que não possuem a expertise em computação.

Complex Networks

Na tradução para o português significa Redes Complexas, é uma abstração de um modelo real, para um modelo computacional, no qual visa exemplificar modelos de relações entre: pessoas, computadores, aeroportos, pontos de acesso e outras tantas abstrações.
