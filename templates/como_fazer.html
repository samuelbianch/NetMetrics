Tutorial

Criando a rede

Para que um grafo possa ser representado, é essencial que o estudo obtenha uma forma de representar estes grafos, mesmo que não de forma visual, mas seja um padrão já utilizado na comunidade. Optou-se então por usar uma lista de arestas (Edge List), que é a representação da adjacência dos vértices em um grafo. Basicamente seu conceito é uma lista que deve conter os vértices numerados por índices, no qual o vértice A aponta para o vértice B.

Arquivo de entrada

Uma lista de arestas no seguinte padrão: sempre o primeiro vértice deve começar com zero, e os vértices devem seguir uma sequência de número inteiros. O arquivo deve ser um .txt

Exemplo:

0 1
0 2
1 2
1 0
2 1

Em termos práticos essa lista de arestas pode ser lida como, o vértice 0 liga com o vértice 1, 0 liga com 2, 1 liga com 2, 1 liga com 0 e 2 liga com 1.

Usando o sistema

1. Clique em "Quero usar!" na página inicial.
2. Clique em "Escolher arquivo" para poder escolher a lista de arestas.
3. Escolha a lista de arestas da sua rede e marque a opção "Direcionada", se ela for uma rede de arestas direcionadas.
4. Por fim, basta apenas fazer as devidas análises sobre a rede, você pode também fazer o download das imagens e do arquivo com dados de saída.
